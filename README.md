# 🌐 Language Translation System using Deep Learning

# Project Overview

This project is a web-based Language Translation System developed using deep learning. It allows users to input English text and translate it into multiple languages such as Hindi, French, and German.

The system uses pretrained Transformer models to generate accurate translations and provides a clean and user-friendly interface.

---

# Objectives

* To develop a real-time language translation system  
* To understand the use of deep learning models  
* To build a web application using Flask  
* To connect frontend and backend  

---

# 🛠️ Technologies Used

* Python  
* Flask (Web Framework)  
* HTML, CSS  
* Hugging Face Transformers  
* PyTorch  

---

# Features

* Multi-language translation (Hindi, French, German)  
* Deep learning-based translation using Transformer models  
* Simple and user-friendly interface  
* Dynamic model loading  
* Real-time translation output  

---

# Working Principle

1. User enters text into the input box  
2. User selects the target language  
3. Request is sent to the Flask backend  
4. The required Transformer model is loaded  
5. The model processes the input text  
6. Translated text is displayed on the screen
