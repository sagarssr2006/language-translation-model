# Language Translation Model using Deep Learning

This project is a simple web-based language translator built using deep learning. It uses Transformer-based models to translate English text into different languages.

 💻 Technologies Used
- Python
- Flask
- Hugging Face Transformers
- PyTorch
- HTML, CSS, JavaScript

 🌍 Features
- Translate English text into:
  - Hindi
  - French
  - German
- Uses deep learning (Transformer model)
- Clean and user-friendly interface
- Custom language selector with flags

 ⚙️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

 📌 Note
- First run may take time as models download.
- Internet is required only for the first run.
